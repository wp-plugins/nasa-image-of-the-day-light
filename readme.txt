**Requirements**

* PHP 5.2.0 or newer

== Installation ==

1. Upload the folder nasa-image-of-the-day-light to the /wp-content/plugins/ directory
2. Activate the plugin NASA Image of the Day through the 'Plugins' menu in WordPress
3. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
